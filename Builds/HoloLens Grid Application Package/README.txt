How to Sideload the build to the HoloLens.
1. Open Device Portal (type in the IP address of the HoloLens into a browser on the same network.)
2. Navigate to "System>Apps" and under Install App select the appx file (Cubes Test_1.0.1.0_Debug_Test/Cubes Test_1.0.1.0_x86_Debug.appxbundle)
3. Hit go to Deploy.

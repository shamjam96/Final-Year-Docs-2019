Main Unity file is in Assets/Main.Unity
Open this file to open the project in Unity.
Building for Android requires the JDK and Android sdk Tools installed.
- Follow Steps here for downloading Android SDK and Java Development Kit 
  https://unity3d.com/learn/tutorials/topics/mobile-touch/building-your-unity-game-android-device-testing
  Then in Unity's build settings, select Android as the platform and press build to built to an apk file.

Current Builds of this application can be found in supporting/Builds
Source Code is in Assets.
